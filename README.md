# Simple Web Crawler App in Typescript

## Description 

This is a simple web crawler implemented with typescript

## Setup

This project uses node.js v24.5.0, it is recommended to use [nvm](https://www.nvmnode.com/guide/introduction.html) to easily swap between node versions. Once nvm is installed, use `nvm use` in the terminal to set node to the correct version.

After this step, you can run the main crawl with npm. For example `npm run crawl`
